The brand's signature status pill — mono, uppercase, stamp radius — mapping gig-day phases to semantic states.

```jsx
<SignalChip tone="load" dot>Load in</SignalChip>
<SignalChip tone="live" hard>Show · Go</SignalChip>
<SignalChip tone="sound" variant="tint">Soundcheck</SignalChip>
<SignalChip tone="stop" variant="outline">Curfew</SignalChip>
```

Tones: `load` (info/teal), `sound` (warning/amber), `doors` (accent/violet), `live` (success/green), `stop` (danger/red), plus `brand` and `ink`. Variants `solid|tint|outline`; `hard` adds the pass shadow.
