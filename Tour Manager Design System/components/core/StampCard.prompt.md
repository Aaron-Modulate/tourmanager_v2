The default Tour Manager surface — flat day-sheet paper, hairline border, optional mono overline tab notched into the top edge.

```jsx
<StampCard overline="Run of show">…</StampCard>
<StampCard tone="stage" halftone>…</StampCard>
<StampCard hard>Laminated-pass look</StampCard>
```

Tones: `paper` (default), `raised` (white), `stage` (dark). `hard` = 2px ink border + offset shadow. `halftone` adds the dot overlay (auto light variant on stage).
